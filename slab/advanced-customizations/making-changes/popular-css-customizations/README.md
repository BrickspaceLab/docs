# Popular CSS customizations

The guides in this section cover the most commonly requested CSS customizations. Each guide provides the exact code snippet to paste into the theme editor's **Custom CSS** field, along with step-by-step instructions.

| Guide | What it changes |
| --- | --- |
| [Change background color](change-background-color.md) | Sets a custom background color on a section |
| [Change heading color](change-heading-color.md) | Changes the text color of headings within a section |
| [Change button color](change-button-color.md) | Overrides the background and text color of buttons |
| [Adjust button size](adjust-button-size.md) | Changes the font size and padding of buttons |
