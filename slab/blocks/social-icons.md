# Social icons


## Overview

A customizable social media icons block that displays a collection of social media links as icons. The block supports various social platforms including TikTok, Instagram, Twitter/X, Facebook, Pinterest, Snapchat, Tumblr, Discord, LinkedIn, Twitch, YouTube, and Vimeo.


## Common use cases

* Display social media links in headers, footers, or sidebars
* Configure social media URLs in theme settings before adding this block
* Adjust icon size and gap spacing based on the design context
