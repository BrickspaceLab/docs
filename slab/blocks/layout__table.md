# Table


## Overview

A table layout container that arranges child blocks in a structured table format with configurable columns. Supports sticky headers, customizable header text, color schemes, and responsive visibility.


## Common use cases

Display structured data in a table format with configurable columns and optional sticky headers.


## Block settings

### Content

| Setting | Description | Options |
|---------|-------------|--------|
| Column headings | Enter a comma-separated list of column headings. E.g. Heading 1, Heading 2, Heading 3 | Text input |

### Color

| Setting | Description | Options |
|---------|-------------|--------|
| Type | Controls the type | • Custom<br>• Base (default) |
| Scheme | Controls the scheme | Color scheme picker |
| Scheme | Controls the scheme | • Body<br>• Accent 1<br>• Accent 2<br>• Accent 3<br>• Shade 1<br>• Shade 2<br>• Shade 3<br>• Blur<br>• Transparent (default) |
| Enable inheritance | Colors will be inherited from the closest parent block that has a set scheme. | Checkbox (default: false) |
| Enable column highlight | Controls the enable column highlight | Checkbox (default: false) |
| Column scheme | Controls the column scheme | • Body<br>• Accent 1<br>• Accent 2<br>• Accent 3<br>• Shade 1 (default)<br>• Shade 2<br>• Shade 3<br>• Blur<br>• Transparent |
| Highlighted column | Select the column to highlight. This column will have a different background color. | 0 – 10 (default: 2) |

### Spacing

| Setting | Description | Options |
|---------|-------------|--------|
| Enable horizontal padding | Controls the enable horizontal padding | Checkbox (default: false) |
| Top spacing | Controls the top spacing | 0 – 300 px (default: 0) |
| Bottom spacing | Controls the bottom spacing | 0 – 300 px (default: 0) |

| Setting | Description | Options |

| Setting | Description | Options |
|---------|-------------|--------|

| Setting | Description | Options |
