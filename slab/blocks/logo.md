# Logo


## Overview

Displays the store logo as either an uploaded image or text fallback. Supports responsive sizing for desktop and mobile devices, customizable typography, alignment, and visibility options.


## Common use cases

- Display your store logo in the header with responsive sizing for desktop and mobile devices.


## Block settings

### Content

| Setting | Description | Options |
|---------|-------------|---------|
| Logo | Upload a logo image | Image picker<br><br>_Visible when show image is enabled_ |
| URL | Link destination when logo is clicked | URL input (default: /) |
| Text | Text to display when no logo image is used | Text input<br><br>_Visible when show image is disabled_ |
| Show image | Toggle between logo image and text display | Checkbox (default: false) |

### Spacing

| Setting | Description | Options |
|---------|-------------|---------|
| Desktop logo height | Height of the logo on desktop devices | 10 – 200 px (default: 24) |
| Mobile logo height | Height of the logo on mobile devices | 10 – 200 px (default: 24) |
| Enable horizontal padding | Adds horizontal padding around the logo | Checkbox (default: false) |

### Style

| Setting | Description | Options |
|---------|-------------|---------|
| Font family | Controls the font family used for text logo | &bull; Standard<br>&bull; Heading (default)<br>&bull; Subheading<br>&bull; Accent<br><br>_Visible when font size contains 'type--' or font size is "custom"_ |
| Font size | Controls the font size of the text logo | &bull; Smaller<br>&bull; Small<br>&bull; Default<br>&bull; Big<br>&bull; Bigger (default)<br>&bull; Heading 1<br>&bull; Heading 2<br>&bull; Heading 3<br>&bull; Heading 4<br>&bull; Heading 5<br>&bull; Heading 6<br>&bull; Custom |
| Size | Custom font size | 4 – 400 px (default: 16)<br><br>_Visible when font size is "custom"_ |
| Line height | Custom line height | 100 – 300 % (default: 150)<br><br>_Visible when font size is "custom"_ |
| Letter spacing | Custom letter spacing | -3 – 3 rem (default: 0)<br><br>_Visible when font size is "custom"_ |

### Layout

| Setting | Description | Options |
|---------|-------------|---------|
| Horizontal alignment | Controls horizontal alignment of the logo | &bull; Left (default)<br>&bull; Center<br>&bull; Right |

### Display

| Setting | Description | Options |
|---------|-------------|---------|
| Visibility | Controls when the block is visible | &bull; All (default)<br>&bull; Mobile only<br>&bull; Desktop only |
