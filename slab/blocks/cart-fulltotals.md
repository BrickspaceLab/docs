# Full subtotal


## Overview

A cart totals block that displays subtotal, discounts, and final total pricing with reactive updates using Alpine.js. The discount row is conditionally shown when discounts are applied.


## Common use cases

- Display a detailed cart pricing breakdown including subtotal, discounts, and total
- Use in cart drawers or cart pages where a comprehensive price summary is needed
- Toggle currency symbol visibility based on store requirements
