# Hotspot


## Overview

This is a banner section that can display hotspot links overlaid on the image.


## Common use cases

* Create an interactive image with clickable hotspot markers
* Build product lookbook images with clickable product spots
* Add interactive location or feature markers to images


## Compatible blocks

The following blocks can be nested within this block:

* [Spot](_g__hotspot-spot.md)


## Block settings

### Content

| Setting | Description | Options |
|---------|-------------|----------|
| Desktop background | 2800 x 1200px recommended. | Image picker |
| Mobile background | 760 x 1010px recommended. | Image picker |
| Show entire image | Shows the entire image within the container | Checkbox (default: false) |

### Spacing

| Setting | Description | Options |
|---------|-------------|----------|
| Minimum height | Minimum height of the block | 0 - 100 % (default: 50 %) |

### Style

| Setting | Description | Options |
|---------|-------------|----------|
| Radius | Controls the border radius | <p>• None<br>• Default (default)<br>• Small<br>• Medium<br>• Large<br>• 1x Large<br>• 2x Large<br>• Full</p> |

### Display

| Setting | Description | Options |
|---------|-------------|----------|
| Visibility | Controls when the block is visible | <p>• Show on all devices (default)<br>• Show on mobile only<br>• Show on desktop only</p> |
