# Input field


## Overview

A flexible form input field block that renders various input types including text, email, phone, and textarea fields. Provides customizable label positioning, placeholder text, validation requirements, and styling options.


## Common use cases

Create customizable form fields for contact forms, newsletter signups, and other data collection needs.


## Block settings

### Content

| Setting | Description | Options |
|---------|-------------|--------|
| Label | Controls the label | Text input |
| Placeholder | Controls the placeholder | Text input |
| Help text | Controls the help text | Rich text input |
| Options | Separate each option with a comma | Textarea input |
| Make this field required | Controls the make this field required | Checkbox (default: false) |

### Display

| Setting | Description | Options |
|---------|-------------|--------|
| Visibility | Controls the visibility | • Show on all devices (default)<br>• Show on mobile only<br>• Show on desktop only |
