# Search input


## Overview

A customizable search form component that provides a responsive search interface with configurable button display, search overlay support, and flexible styling options.


## Common use cases

* Add a search interface to headers, overlays, or search pages
* Enable search overlay for an immersive search experience
* Configure button visibility and styling to match your theme design


## Block settings

### Content

| Setting | Description | Options |
|---------|-------------|---------|
| Placeholder | Controls the placeholder | Text input |
| Enable search overlay | Controls the enable search overlay | Checkbox (default: true) |
| Show button | Controls the show button | Checkbox (default: false) |
| Button label | Controls the button label | <p>Text input<br><br>Visible when show button is true</p> |
| Type | Controls the type | <p>• Custom<br>• Base (default)<br><br>Visible when show button is true</p> |
| Button | Controls the button | <p>Color scheme picker<br><br>Visible when show button is true and color type is custom</p> |
| Button | Controls the button | <p>• Plain (default)<br>• Main<br>• Blur<br>• Outline<br>• Inverted outline<br>• Transparent<br>• Inverted transparent<br><br>Visible when show button is true and color type is base</p> |
| Layout | Controls the layout | <p>• Stacked<br>• Inline (default)<br><br>Visible when show button is true</p> |
