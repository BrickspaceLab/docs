# Filter


## Overview

A comprehensive filtering system for collection and search pages that provides sorting options and advanced filtering capabilities. Supports multiple filter types including boolean filters, list filters, and price range sliders with an accordion-style interface.


## Common use cases

Add sorting and filtering controls to collection and search pages for improved product discovery.
