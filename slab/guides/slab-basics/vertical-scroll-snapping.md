# Vertical scroll snapping

Vertical scroll snapping lets sections lock into place as customers scroll through the storefront. Use it when you want a more guided, section-by-section browsing experience.



## How to use scroll snapping

Vertical scroll snapping has two parts.

{% stepper %}
{% step %}
### Enable scroll snapping globally

1. Open the theme editor.
2. Go to **Theme settings**.
3. Open **Layout**.
4. Enable **Enable scroll snapping**.

Once this is enabled, supported sections will show the **Scroll snap alignment** setting.
{% endstep %}

{% step %}
### Set scroll snap alignment on sections

1. Go back to the section you want to control.
2. Find **Scroll snap alignment**.
3. Choose the alignment you want for that section.

If a section is left on **None**, it will keep scrolling normally.

Repeat this for each section that should participate in scroll snapping.
{% endstep %}
{% endstepper %}
